NAND Gate States

G1: NAND(T, CLK, Q') → S'
S' = 1
G2: NAND(T, CLK, Q) → R'
R' = 1
G3: NAND(S', Q') → Q
Q  = 0
G4: NAND(R', Q) → Q'
Q' = 1
G1,G2 = 3-input NAND (Toggle control)
G3,G4 = SR Latch (memory core)
⟳ Q feeds G2  ·  Q' feeds G1
✓ T=0 → HOLD  ·  T=1 → TOGGLE
