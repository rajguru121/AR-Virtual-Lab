NAND Gate States

G1: NAND(S, CLK)
S' = 1
G2: NAND(R, CLK)
R' = 1
G3: NAND(S', Q') → Q
Q = 0
G4: NAND(R', Q) → Q'
Q' = 1
G1,G2 = Input control stage (2-input)
G3,G4 = SR Latch (memory core)
⚠ S=R=1 is FORBIDDEN
