XNOR Gate (Exclusive NOR) outputs HIGH (1) when both inputs are the same — both 0 or both 1. It is the complement of the XOR gate and is also called an equivalence gate.

Boolean Expression: Y = A ⊙ B = (A · B) + (A̅ · B̅)
